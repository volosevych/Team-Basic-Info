## Description:
![License](https://img.shields.io/badge/License--blue.svg "License Badge")
    The application prompts the user for information about the team manager and then information about the team members. The user can input any number of team members, and they may be a mix of engineers and interns. When the user has completed building the team, the application will create an HTML file with all added information.
